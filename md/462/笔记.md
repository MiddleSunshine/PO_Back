- NAME 名字
- SYNOPSIS 概述
- DESCRIPTION 描述
- OPTIONS 参数
- EXIT STATUS 异常退出状态
- RETURN VALUE 返回值
- ERRORS 错误
- ENVIRONMENT 环境
- FILES 关联文件
- VERSIONS 版本
- CONFIRMING TO 符合标准
- NOTES 备注
- EXAMPLE 例子
- AUTHORS 作者
