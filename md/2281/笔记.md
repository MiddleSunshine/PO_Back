   
```c

#define _BSD_SOURCE
#include <unistd.h>

char *getpass(const char *prompt);

// Returns pointer to statically allocated input password string on success,or NULL on error.

```

这个函数的本质作用就是读取用户输入的密码，为了起到这个作用，比起传统的标准输入，它有以下这些作用：

1. 阻隔了 `ctrl+c` 这样的操作
2. 关闭回显，即输入密码的时候显示器上是没有显示的
3. 结果中自动过滤掉最后的换行符
