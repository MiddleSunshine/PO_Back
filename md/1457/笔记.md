```python
data = [value * 2 for value in range(1, 10)]

print(data)
print(data[0:3])
print(data)
print(data[-3:-1])
print(data[1:])
```

```bash
[2, 4, 6, 8, 10, 12, 14, 16, 18]
[2, 4, 6]
[2, 4, 6, 8, 10, 12, 14, 16, 18]
[14, 16]
[4, 6, 8, 10, 12, 14, 16, 18]
```

上面代码的注意事项：

1. 切割列表是右闭合的，即`[0:3)`
2. 只是取出列表的部分值，但是列表本身不会发生变化
3. 支持负数索引
4. 右侧的值支持忽略，这样就会获取值直到最后一个元素
