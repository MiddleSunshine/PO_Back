定义在`limits.h`中，代表了一个用户所能所属的最多的组的数量

```c
#include <unistd.h>
#include <limits.h>
#include <stdio.h>

int main(int argc, char const *argv[])
{
    gid_t groupList[NGROUPS_MAX+1];
    getgroups(NGROUPS_MAX,groupList);
    for (int i = 0; i <= NGROUPS_MAX; i++)
    {
        printf("%d\n",groupList[i]);
    }
    return 0;
}
```
